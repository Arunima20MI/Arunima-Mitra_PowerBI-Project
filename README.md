# 🚦 Road Accident Analysis Dashboard

An interactive **Business Intelligence (BI) dashboard** built using **Microsoft Power BI** to analyze road accident data and extract meaningful insights for improving road safety.

---

## 📌 Project Overview

Road accidents are a major global issue, causing significant loss of life and property. This project focuses on transforming raw accident data into a **visual and interactive dashboard** that helps identify patterns, trends, and high-risk factors.

The dashboard provides insights into:

* Accident severity
* Vehicle type involvement
* Road and environmental conditions
* Yearly and monthly trends

---

## 🎯 Objectives

* Analyze road accident data effectively
* Identify high-risk factors and accident-prone conditions
* Compare accident trends across different time periods
* Support data-driven decision-making for road safety

---

## 📊 Dashboard Features

* **KPI Metrics**

  * Total Accidents
  * Total Casualties
  * Fatal, Serious, and Slight Casualties

* **Vehicle Type Analysis**

  * Casualties by Cars, Bikes, Vans, Buses, etc.

* **Time-Based Trends**

  * Monthly comparison (Current Year vs Previous Year)

* **Geographical Insights**

  * Urban vs Rural accident distribution

* **Road Type Analysis**

  * Single carriageway, dual carriageway, roundabouts, etc.

* **Environmental Factors**

  * Light conditions (Day/Night)
  * Weather and road surface filters

---

## 🛠️ Tech Stack

* **Visualization Tool:** Microsoft Power BI
* **Dataset:** CSV (Road Accident Data)
* **Data Processing:** Power Query Editor
* **Calculations:** DAX (Data Analysis Expressions)

## 📈 Key Insights

* 📉 Overall accidents and casualties decreased (~12%)
* ⚠️ Fatalities reduced significantly (~33%)
* 🚗 Cars contribute the highest number of accidents
* 🌆 Urban areas show higher accident rates than rural areas
* 🛣️ Single carriageways are the most accident-prone roads
* 🌞 Majority of accidents occur during daytime
